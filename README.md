## Project Name : Summer Store

## About SummerStore Project  : our Smart Summer Shopping Partner

SummerStore is a modern summer-focused eCommerce platform where users can explore and purchase seasonal products like sunglasses, skincare items, summer outfits, beach accessories, caps, sandals, and more. The project is designed with a fresh summer vibe, responsive layout, and smooth user experience. Users can browse products from JSON data, view popular products, check product details after login, and manage their profile through authentication using BetterAuth.

## 3 Features About SummerStore Project :

1. Protected Product Details Page: Users can browse products freely, but product details are protected. If a user is not logged in, they are redirected to the login page. After successful login, they can access the product details page.

2. BetterAuth Authentication System: SummerStore includes login, register, and Google social login using BetterAuth. Users can register with name, email, photo URL, and password, then login securely and access protected routes.

3. User Profile & Update Information : Logged-in users can visit the My Profile page to see their name, email, and profile photo. They can also update their name and image from a separate update profile route.
